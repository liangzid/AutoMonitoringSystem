这是使用大创行人重识别算法开发的实用系统。

# 在一台陌生的电脑上安装配置pyqt5
安装pyqt5
```shell
sudo apt-get install python3-pyqt5
```
安装qtdesigner
```shell
sudo apt-get install qt5-default qttools5-dev-tools
```
